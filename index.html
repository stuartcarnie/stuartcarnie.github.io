Objective-C: ternary operator

Mar 15, 2012 · 2 minute read
tipsobjective-c The ternary operator, also known as a conditional expression is a C construct. What follows is an example used for conditional assignment: result = condition_expression ? true_expression : false_expression; If condition_expression evaluates to true, result will be assigned the true_expression; otherwise, result will be assigned the false_expression. A GNU extension to the ternary operator, also available in Objective-C, is the ability to omit the true_expression as follows: result = first_expression ?: second_expression; result will be assigned the value of first_expression if it evaluates to true or second_expression if first_expression evaluates to false.
